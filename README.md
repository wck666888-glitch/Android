# TV IR Remote Simulator

## 项目背景
TV嵌入式实习项目 - TV IR模拟器设计&实现。

## Git 提交规范
本项目执行严格的 Git Commit Log 规范。提交时请遵循以下格式：

```
[关键字] 英文简单描述 (空一行)
[what] 详细说明修改什么问题
[why] 详细说明问题产生原因和(或)为什么做这个修改
[how] 详细说明是怎么修改的
```

### 关键字说明
- **bugfix**: 针对 BUG 的修改和完善
- **feature**: 功能和需求的修改和完善
- **config**: 客户配置和参数的修改
- **merge**: 合并其他分支
- **revert**: 还原提交

## 开发环境
- Git 版本: 2.52.0.windows.1 (安装于 D:\Git)
- 语言: Android / Java
